# 🍱 便當訂餐系統 (Bento Order System)

一個基於 **Blazor Server** 與 **MudBlazor** 開發的便當訂購解決方案。旨在簡化每日點餐流程，並提供管理員統計與調度功能。

## 🌟 核心功能

-   **視覺化日曆訂餐**：直觀的月曆介面，支援每日雙餐點（主餐、加點）選擇。
-   **自動化截止機制**：每日 **09:00** 自動鎖定當日訂單，防止誤改與漏訂。
-   **管理員控管中心**：
    -   **每日統計**：一鍵查看各類餐點總數，方便與廠商核對。
    -   **詳細管理**：管理者可查看每位使用者的訂餐狀態（已訂/未訂）。
    -   **強制調度**：管理員擁有最高權限，可無視日期與 09:00 限制，隨時補登或修正所有人的訂單。
    -   **週期結算**：支援自定義日期區間（如上月 26 至本月 25）的總訂餐量統計。
-   **響應式設計**：完美適配桌面端介面（MudBlazor 驅動）。

## 🛠️ 技術棧

-   **Framework**: .NET 10 / Blazor Server
-   **UI Library**: MudBlazor
-   **ORM**: Entity Framework Core
-   **Database**: SQLite
-   **State Management**: GlobalState Pattern
